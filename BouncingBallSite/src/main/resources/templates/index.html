Bouncing Balls

by Emie Watin Abalora

Bouncing Ball project implements Runnable so that each ball can possess its thread that helps with creating/starting/managing/stopping threads in the program. Each ball maintains its attributes such as size, coordinates, color, velocity, and boundary limits. Accessor and Mutator methods are available for each respective attribute to allow BallPanel to maintain a Ball's creation and positioning.

Ball class

Ball class implements Runnable, it has a method that updates the position of the ball, detects boundary collision, and updates ball velocity.

BallPanel class

BallPanel extends Jpanel, which has a method that creates ball bounce and executes it in a thread; for each mouse click. It randomly generates a ball color, adds a new ball object to a LinkedList, and runs a new thread of created ball object.

BouncingBall class

Bouncing Ball contains the main class; it's the core method of the program and calls all others. It can't return values and accepts parameters for complex command-line processing.

Student_Sig_Block class

Student segnature class contains a variables to create ascii border for signature block and also declare and initialize variables to hold student/course information.
